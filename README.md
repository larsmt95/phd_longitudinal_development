# Paper 3 Analysis

Dette prosjektet analyserer og visualiserer longitudinelle data, med fokus på fysisk utvikling hos yngre lagidrettsutøvere.

## Struktur

- `data/raw/` : Rådata som sporet i GitHub
- `scripts/` : Alle R-skript for datahåndtering, analyse og visualisering
- `plots/` : Visualiseringer generert fra skriptene

## Hvordan bruke prosjektet

1. Åpne RStudio prosjektfil `paper_3.Rproj`  
2. Kjør skriptene i `scripts/` i rekkefølge:
   - `01_data_prep.R` : Rydding og transformasjon av rådata  
   - `02_tables.R` : Tabeller for analyse  
   - `03_models_mixed.R` : Mixed models  
   - … videre skript for plots og analyser

## Bidrag

For å bidra, klon repoet, lag endringer og push via GitHub.
